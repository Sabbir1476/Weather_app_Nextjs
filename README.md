# 🌦️ Weather App (Next.js)

🚀 Live Demo: https://weather-app-nextjs-five.vercel.app/

A modern and responsive weather application built using **Next.js** that allows users to check real-time weather information for any city.

---

## ✨ Features

* 🌍 Search weather by city name
* 📍 Auto-detect user location (if enabled)
* 🌡️ Current temperature, feels-like, and conditions
* 📅 Daily & hourly forecast
* 💨 Wind speed, humidity, and other details
* ⚡ Fast and responsive UI
* 🎨 Clean design with smooth user experience

> The app uses weather APIs to fetch real-time data including temperature, wind, and forecast details. ([GitHub][1])

---

## 🛠️ Tech Stack

* **Frontend:** Next.js, React
* **Styling:** Tailwind CSS
* **API:** Weather API (e.g., Open-Meteo / OpenWeather)
* **Deployment:** Vercel

---

## ⚙️ Getting Started

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/your-repo-name.git
```

### 2️⃣ Install dependencies

```bash
npm install
```

### 3️⃣ Run the development server

```bash
npm run dev
```

### 4️⃣ Open in browser

```
http://localhost:3000
```

---

## 🔑 Environment Variables

Create a `.env.local` file and add:

```
NEXT_PUBLIC_API_KEY=your_api_key_here
```

---

## 🚀 Deployment

This project is deployed on **Vercel** for fast and seamless hosting.

To deploy your own:

1. Push your code to GitHub
2. Import project into Vercel
3. Add environment variables
4. Deploy 🎉
